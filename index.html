h1 à h6

(Voltar)

Representam os níveis de título de uma seção, sendo "h1" é o nível mais alto e "h6" o mais baixo.

p

(Voltar)

É usado para representar um parágrafo.

mark

(Voltar)

Marca o texto em amarelo, destacando-o.

small

(Voltar)

Diminue o tamaho da fonte do texto por padrão

i

(Voltar)

Trás o texto em itálico

u

(Voltar)

Trás o texto sublinhado

strong

(Voltar)

Esse elemento HTML dá ao texto uma forte importância, e o texto é mostrado em negrito.

ol

(Voltar)

Representa uma lista de items ordenada. Uma lista numerada como padrão

ul

(Voltar)

Representa uma lista de items não ordenada. Uma lista que trás um ponto como padrão de ordenação

li

(Voltar)

É usado para representar um item de uma lista.

a

(Voltar)

Juntamente com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.

hr

(Voltar)

Apresenta uma linha horizontal, mas semanticamente representa uma quebra temática entre elementos de nível de parágrafo.

sub

(Voltar)

Usando esse elemento, você pode diminuir o tamanho da fonte e deixando-o subscrito

sup

(Voltar)

Usando esse elemento,você pode diminuir o tamanho da fonte e deixando-o sobrescrito.

blockquote

(Voltar)

É usado para representar uma citação na página.

font

(Voltar)

Esse recurso não é mais recomendado. Embora alguns navegadores ainda possam suportá-lo, ele pode já ter sido removido dos padrões da Web relevantes, pode estar em processo de eliminação ou pode ser mantido apenas para fins de compatibilidade.

del

(Voltar)

Representa um intervalo de texto que foi excluído de um documento.

abbr

(Voltar)

O elemento representa uma abreviação de uma palavra ou acrônimo.
